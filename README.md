# bank-management-system-using-Cpp
This C++ project demonstrates a strong grasp of Object-Oriented Programming (OOP) and Data Persistence, which are key competencies for campus placements.

Project Description
Project Name: Automated Banking Management System with File Persistence
Description: Developed a robust, console-based banking application in C++ utilizing Object-Oriented Programming (OOP) principles like inheritance and exception handling. The system features a dual-interface for Admins (account unblocking/deactivation) and Users (fund transfers, PIN management, and transaction history). It ensures data persistence across sessions by implementing File Handling to store binary record data and transaction logs.

Key Technical Highlights:-

Encapsulation & Inheritance: Structured code using classes (user, account, error) to modularize sensitive data and banking logic.

File I/O Operations: Integrated fstream to manage binary data storage (.bin) and generated human-readable transaction reports (.txt).

Security & Validation: Implemented a multi-tier security system including 4-digit PIN authentication with an auto-block feature after three failed attempts and custom exception handling for invalid transactions.

Transaction Logic: Built a real-time transfer module that calculates cashbacks and maintains a chronological record of credits/debits using the <ctime> library.
